# 스프링 부트 학습을 위한 저장소

### 상세설명

* 하나의 저장소에 여러개의 프로젝트를 묶어 관리 하는 저장소 입니다.
* 코드를 치며 이론을 익혀나가기 위한 목적으로 개설(Top->Down)


### 개발환경

* OS : Windows
* Tools : InteliJ
* JDK Version : 11
* DB : H2 or Mysql Workbench
